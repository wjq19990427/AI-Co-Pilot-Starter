## 层级结构

```
{层C}（UI / 接口层）
    ↓ 依赖
{层B}（业务逻辑层）
    ↓ 依赖
{层A}（基础设施层）
```

反向依赖 = 架构违规，禁止跨层直接调用内部实现。
